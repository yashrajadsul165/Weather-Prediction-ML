# Weather Prediction using Machine Learning

- Data Cleaning
- Data Visualization
- Model Training
- Weather Prediction
- Performance Evaluation

## Technologies Used

- Python
- Pandas
- NumPy
- Matplotlib
- Scikit-learn
- Jupyter Notebook

## Project Workflow

1. Collect Dataset
2. Clean Data
3. Train Model
4. Test Model
5. Predict Weather
